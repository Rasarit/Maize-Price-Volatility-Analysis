# Maize Price Volatility Analysis

## Project Overview

This project analyzes **volatility in maize prices** using historical market price data in **R**.

Price volatility represents how much prices fluctuate over time. Understanding volatility helps market participants identify periods of higher market instability and price risk.

---

## Dataset

The dataset contains historical maize market prices.

| ColumnDescription |                                    |
| ----------------- | ---------------------------------- |
| date              | Date of observation                |
| price             | Maize market price (₹ per quintal) |

The dataset includes **monthly price observations over several years**.

---

## Tools Used

* **R**
* **RStudio**
* **ggplot2** – data visualization
* **dplyr** – data manipulation

---

## Project Structure

```
maize-price-volatility
│
├── maize_price_arrivals.csv
├── price_volatility_analysis.R
├── maize_price_change.svg
├── maize_price_volatility.svg
└── README.md

```

---

## 1. Monthly Price Change

This chart shows the **monthly change in maize prices**.
It highlights how much prices increase or decrease between consecutive months.

---

## 2. Price Volatility

This chart shows the **percentage change in maize prices**, which helps measure price volatility over time.

Periods with larger fluctuations indicate **higher market volatility**.

---

## Methodology

The analysis followed these steps:

1. Load maize price data
2. Calculate monthly price changes
3. Compute percentage price changes
4. Visualize price fluctuations using line charts
5. Measure price volatility using standard deviation

---

## Key Insights

### Insight 1 — Price Fluctuations

Maize prices show periodic fluctuations over time, reflecting changes in market conditions.

### Insight 2 — Market Volatility

Some periods exhibit larger price movements, indicating higher market volatility.

### Insight 3 — Risk Awareness

Understanding price volatility can help farmers, traders, and agri-business firms better anticipate market risks.

---

## How to Run the Project

1. Clone this repository
2. Open the project in **RStudio**

Install required packages:

```
install.packages("ggplot2")
install.packages("dplyr")

```

Run the script:

```
price_volatility_analysis.R

```
